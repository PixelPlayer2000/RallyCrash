RallyCrash README

This is a small rally racing game.

It is developed by PixelPlayer2000 and Gobledegook.

How to play: Download the zip from https://github.com/PixelPlayer2000/RallyCrash/archive/master.zip, unzip it and open "RallyCrash.exe".

Key bindings:
    accelerate - up key
    decelerate - down key
    hand brake - space bar
    shift up - 'W' key
    shift down - 'S' key
    shift into park - 'P' key
    steer right - right key
    steer left - left key
    quit to menu - 'esc' key
